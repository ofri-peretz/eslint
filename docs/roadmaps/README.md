# `docs/roadmaps/` — Long-form Rule & Feature Roadmaps

Multi-quarter roadmaps for major rule families. These complement [`../../ROADMAP.md`](../../ROADMAP.md), which holds the high-level phased plan; the docs here go deep on a single domain.

## Current roadmaps

- [`owasp-agentic-eslint-roadmap.md`](./owasp-agentic-eslint-roadmap.md) — full ESLint rule plan for **OWASP Top 10 for Agentic Applications 2026**. Drives `eslint-plugin-vercel-ai-security` and adjacent agentic rules.
- [`owasp-llm-2025-eslint-rules-roadmap.md`](./owasp-llm-2025-eslint-rules-roadmap.md) — rule plan for **OWASP Top 10 for LLM Applications 2025**.

## Conventions

- One roadmap per OWASP framework version. Bump the file when the framework version bumps.
- Cross-link to `../../CLAIMS.md` for any claim derived from the roadmap.
- Cross-link to research notes in [`../research/`](../research/) when the roadmap relies on external sources.
