## Reporting a Vulnerability

We take security issues seriously. If you discover a security vulnerability in this project, please **DO NOT** open a public issue.

Instead, please report it privately:

1.  Email **ofri@snappy.com** (or your preferred security contact email).
2.  In your report, please include:
    - A description of the vulnerability.
    - Steps to reproduce the issue.
    - The version of the package you are using.
    - Any relevant code snippets or configurations.

We will acknowledge your report within 48 hours and work with you to address the issue. We appreciate your responsible disclosure and will give credit to researchers who help us improve security.
