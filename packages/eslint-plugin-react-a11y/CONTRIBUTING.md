# Contributing to eslint-plugin-react-a11y

Thank you for your interest in contributing to `eslint-plugin-react-a11y`!

## Quick Start

```bash
# Clone the monorepo
git clone https://github.com/ofri-peretz/eslint.git
cd eslint

# Install dependencies
pnpm install

# Build this package
nx build eslint-plugin-react-a11y

# Run tests
nx test eslint-plugin-react-a11y
```
