# Contributing to eslint-plugin-react-features

Thank you for your interest in contributing to `eslint-plugin-react-features`!

## Quick Start

```bash
# Clone the monorepo
git clone https://github.com/ofri-peretz/eslint.git
cd eslint

# Install dependencies
npm install

# Build this package
nx build eslint-plugin-react-features

# Run tests
nx test eslint-plugin-react-features
```
