## Overview

`eslint-plugin-react-best-practices` enforces React component patterns and conventions.

## Installation

```bash tab="npm"
npm install eslint-plugin-react-best-practices
```

```bash tab="pnpm"
pnpm add eslint-plugin-react-best-practices
```

## Configuration

```js title="eslint.config.js"
import reactBestPractices from 'eslint-plugin-react-best-practices';

export default [reactBestPractices.configs.recommended];
```
