## Overview

`eslint-plugin-react-performance` helps identify performance issues in React applications.

## Installation

```bash tab="npm"
npm install eslint-plugin-react-performance
```

```bash tab="pnpm"
pnpm add eslint-plugin-react-performance
```

## Configuration

```js title="eslint.config.js"
import reactPerformance from 'eslint-plugin-react-performance';

export default [reactPerformance.configs.recommended];
```
