## Overview

`eslint-plugin-module-federation` enforces micro-frontend architecture patterns.

## Installation

```bash tab="npm"
npm install eslint-plugin-module-federation
```

```bash tab="pnpm"
pnpm add eslint-plugin-module-federation
```

## Configuration

```js title="eslint.config.js"
import moduleFederation from 'eslint-plugin-module-federation';

export default [moduleFederation.configs.recommended];
```
