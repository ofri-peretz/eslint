## Overview

`eslint-plugin-naming-conventions` enforces consistent naming patterns.

## Installation

```bash tab="npm"
npm install eslint-plugin-naming-conventions
```

```bash tab="pnpm"
pnpm add eslint-plugin-naming-conventions
```

## Configuration

```js title="eslint.config.js"
import namingConventions from 'eslint-plugin-naming-conventions';

export default [namingConventions.configs.recommended];
```
