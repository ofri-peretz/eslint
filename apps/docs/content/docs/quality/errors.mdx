## Overview

`eslint-plugin-errors` enforces robust error handling patterns.

## Installation

```bash tab="npm"
npm install eslint-plugin-errors
```

```bash tab="pnpm"
pnpm add eslint-plugin-errors
```

## Configuration

```js title="eslint.config.js"
import errors from 'eslint-plugin-errors';

export default [errors.configs.recommended];
```
