## Overview

`eslint-plugin-react-hooks-best-practices` enforces React hooks best practices.

## Installation

```bash tab="npm"
npm install eslint-plugin-react-hooks-best-practices
```

```bash tab="pnpm"
pnpm add eslint-plugin-react-hooks-best-practices
```

## Configuration

```js title="eslint.config.js"
import reactHooks from 'eslint-plugin-react-hooks-best-practices';

export default [reactHooks.configs.recommended];
```
