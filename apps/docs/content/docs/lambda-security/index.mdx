## Installation

```bash
npm install --save-dev eslint-plugin-lambda-security
```

## Configuration

```js title="eslint.config.js"
import lambdasecurity from 'eslint-plugin-lambda-security';

export default [lambdasecurity.configs.recommended];
```

## Available Presets

| Preset | Description |
|--------|-------------|
| `recommended` | Balanced security for most projects |
| `strict` | Maximum enforcement (all rules as errors) |
