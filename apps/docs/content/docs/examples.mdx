## Get Started in 30 Seconds

<Steps>
  <Step>
    ### Install
    ```bash
    npm install --save-dev eslint-plugin-secure-coding
    ```
  </Step>
  <Step>
    ### Configure
    ```js title="eslint.config.js"
    import secureCoding from 'eslint-plugin-secure-coding';
    
    export default [
      secureCoding.configs.recommended,
    ];
    ```
  </Step>
  <Step>
    ### Run
    ```bash
    npx eslint .
    ```
  </Step>
</Steps>
