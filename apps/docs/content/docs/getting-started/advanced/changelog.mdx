## Version History

This page aggregates changelog information from all ESLint Interlace plugins, fetched dynamically from GitHub.

<Callout type="info" title="Live Data">
  Changelog data is fetched directly from GitHub CHANGELOG.md files and cached
  for 2 hours. Updates to the repository are reflected automatically‚Äîno
  redeployment needed.
</Callout>

---

## How It Works

Each plugin maintains its own `CHANGELOG.md` in its package directory. This page aggregates them in real-time:

1. **GitHub Raw Content** ‚Äî Files are fetched from `raw.githubusercontent.com`
2. **2-Hour Cache** ‚Äî Data is cached using our json-cache policy
3. **Parsed Entries** ‚Äî Markdown is parsed to extract version, date, and type

## API Access

You can programmatically access changelog data via our API:

```bash
# Get all changelogs (parsed)
curl https://interlace.dev/api/changelog

# Get specific plugin changelog
curl https://interlace.dev/api/changelog?plugin=jwt

# Get raw markdown
curl https://interlace.dev/api/changelog?plugin=jwt&raw=true
```

### Response Format

```json
{
  "success": true,
  "data": {
    "plugin": "jwt",
    "path": "packages/eslint-plugin-jwt/CHANGELOG.md",
    "entries": [
      {
        "version": "1.4.0",
        "date": "2026-01-15",
        "type": "feature",
        "content": "Added no-none-algorithm rule..."
      }
    ],
    "fetchedAt": "2026-01-31T12:00:00Z"
  },
  "meta": {
    "source": "github",
    "ttl": 7200
  }
}
```
