## What is ESLint Interlace?

ESLint Interlace is a comprehensive ecosystem of **security and quality ESLint plugins** designed to protect your JavaScript and TypeScript applications from vulnerabilities while enforcing best practices.

<Callout type="info" title="Quick Start">
  Install the complete suite with a single command: `npm install
  eslint-config-interlace`
</Callout>

## Quick Setup Guide

<Steps>

### Install the package

```bash tab="npm"
npm install eslint-config-interlace
```

```bash tab="pnpm"
pnpm add eslint-config-interlace
```

```bash tab="yarn"
yarn add eslint-config-interlace
```

### Configure ESLint

Create `eslint.config.js` in your project root:

```js title="eslint.config.js"
import interlace from 'eslint-config-interlace';

export default [...interlace.configs.recommended];
```

### Run the linter

```bash
npx eslint .
```

</Steps>
