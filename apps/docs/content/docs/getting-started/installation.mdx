## Quick Start

Install ESLint and your first security plugin. Choose your package manager:

### npm

```bash tab="secure-coding"
npm install eslint eslint-plugin-secure-coding --save-dev
```

```bash tab="browser-security"
npm install eslint eslint-plugin-browser-security --save-dev
```

```bash tab="jwt"
npm install eslint eslint-plugin-jwt --save-dev
```

```bash tab="node-security"
npm install eslint eslint-plugin-node-security --save-dev
```

### pnpm

```bash tab="secure-coding"
pnpm add eslint eslint-plugin-secure-coding -D
```

```bash tab="browser-security"
pnpm add eslint eslint-plugin-browser-security -D
```

```bash tab="jwt"
pnpm add eslint eslint-plugin-jwt -D
```

```bash tab="node-security"
pnpm add eslint eslint-plugin-node-security -D
```

### yarn

```bash tab="secure-coding"
yarn add eslint eslint-plugin-secure-coding --dev
```

```bash tab="browser-security"
yarn add eslint eslint-plugin-browser-security --dev
```

```bash tab="jwt"
yarn add eslint eslint-plugin-jwt --dev
```

```bash tab="node-security"
yarn add eslint eslint-plugin-node-security --dev
```

Then create `eslint.config.js`:

```js title="eslint.config.js"
import secureCoding from 'eslint-plugin-secure-coding';

export default [secureCoding.configs.recommended];
```
