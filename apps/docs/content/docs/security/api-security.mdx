## Overview

`eslint-plugin-api-security` enforces secure REST API patterns.

## Installation

```bash tab="npm"
npm install eslint-plugin-api-security
```

```bash tab="pnpm"
pnpm add eslint-plugin-api-security
```

## Configuration

```js title="eslint.config.js"
import apiSecurity from 'eslint-plugin-api-security';

export default [apiSecurity.configs.recommended];
```
