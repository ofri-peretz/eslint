## Overview

`eslint-plugin-browser-security` provides 12 rules for protecting client-side JavaScript applications from common web vulnerabilities.

## Installation

```bash tab="npm"
npm install eslint-plugin-browser-security
```

```bash tab="pnpm"
pnpm add eslint-plugin-browser-security
```

```bash tab="yarn"
yarn add eslint-plugin-browser-security
```

## Configuration

```js title="eslint.config.js"
import browserSecurity from 'eslint-plugin-browser-security';

export default [browserSecurity.configs.recommended];
```
