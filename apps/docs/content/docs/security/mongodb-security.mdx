## Overview

`eslint-plugin-mongodb-security` prevents NoSQL injection and enforces secure MongoDB patterns.

## Installation

```bash tab="npm"
npm install eslint-plugin-mongodb-security
```

```bash tab="pnpm"
pnpm add eslint-plugin-mongodb-security
```

## Configuration

```js title="eslint.config.js"
import mongodbSecurity from 'eslint-plugin-mongodb-security';

export default [mongodbSecurity.configs.recommended];
```
