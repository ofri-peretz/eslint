## Overview

`eslint-plugin-crypto` enforces secure cryptographic practices in JavaScript applications.

## Installation

```bash tab="npm"
npm install eslint-plugin-crypto
```

```bash tab="pnpm"
pnpm add eslint-plugin-crypto
```

## Configuration

```js title="eslint.config.js"
import crypto from 'eslint-plugin-crypto';

export default [crypto.configs.recommended];
```
