## Overview

`eslint-plugin-jwt` provides rules for secure JWT handling and validation.

## Installation

```bash tab="npm"
npm install eslint-plugin-jwt
```

```bash tab="pnpm"
pnpm add eslint-plugin-jwt
```

## Configuration

```js title="eslint.config.js"
import jwt from 'eslint-plugin-jwt';

export default [jwt.configs.recommended];
```
