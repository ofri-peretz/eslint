## Overview

`eslint-plugin-secrets` detects hardcoded secrets, API keys, and credentials in code.

## Installation

```bash tab="npm"
npm install eslint-plugin-secrets
```

```bash tab="pnpm"
pnpm add eslint-plugin-secrets
```

## Configuration

```js title="eslint.config.js"
import secrets from 'eslint-plugin-secrets';

export default [secrets.configs.recommended];
```
