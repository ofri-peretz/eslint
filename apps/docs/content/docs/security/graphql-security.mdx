## Overview

`eslint-plugin-graphql-security` protects GraphQL operations from common vulnerabilities.

## Installation

```bash tab="npm"
npm install eslint-plugin-graphql-security
```

```bash tab="pnpm"
pnpm add eslint-plugin-graphql-security
```

## Configuration

```js title="eslint.config.js"
import graphqlSecurity from 'eslint-plugin-graphql-security';

export default [graphqlSecurity.configs.recommended];
```
