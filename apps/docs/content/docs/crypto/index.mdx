## Installation

```bash
npm install --save-dev eslint-plugin-crypto
```

## Configuration

```js title="eslint.config.js"
import crypto from 'eslint-plugin-crypto';

export default [crypto.configs.recommended];
```

## Available Presets

| Preset | Description |
|--------|-------------|
| `recommended` | Balanced security for most projects |
| `strict` | Maximum enforcement (all rules as errors) |
